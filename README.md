# Projeto-Árvore

Plantando árvores para colher os frutos.

Este repositório é a trilha de estudos para o curso de **Sistemas para Internet/Universidade Federal de Lavras**, com foco em **programação multiparadigma** — unindo Orientação a Objetos, Programação Funcional, arquitetura de sistemas e sustentabilidade digital. Aqui registro leituras, reflexões, projetos e experimentos enquanto cultivo as bases para um futuro profissional alinhado com transparência, resiliência e impacto positivo.
